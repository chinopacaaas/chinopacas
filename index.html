Propiedad Position

Especifica el tipo de método de posicionamiento, hay cinco valores de propiedad; top, bottom, left, rigth y z-index.
Static: los elementos en HTML tienen por defecto esta propiedad, estos no se ven afectados por las propiedades antes mencionadas, siempre se posiciona con el flujo normal de la página.
Relative: el elemento se desplaza respecto a su posición establecida mediante el posicionamiento original, se controlan mediante top, bottom, left y rigth. Los demás elementos no se desplazan a ocupar el lugar que deja.
Absolute: se emplea para establecer de forma exacta el posicionamiento del elemento, cuando un elemento se posiciona de forma absoluta los demás elementos se ven afectados y estos ocuparan el posicionamiento que dejo el elemento.
Fixed: está ubicado con relación a la ventana gráfica. Este elemento no se desplaza cuando la página lo hace y no deja espacio en la página como normalmente lo haría, este se establece con respecto de la ventana gráfica.
Sticky: se posiciona con respecto al desplazamiento del usuario y esta alterna entre relativo o fijo, se posiciona de manera relativa hasta que encuentra una posición de desplazamiento dada, pasa a una posición fija.

Notas

Es importante recordar que los valores de propiedad, dependiendo de la propiedad, hacen o no efecto sobre los elementos. Solo los elementos relative son manipulados por los valores de propiedad.

Resumen

Especifica el metodo de posicionamiento del elemento, variando en funcion de la propiedad. Los elementos llevan por default la propiedad static y esta hace que no tengan efecto los valores de propiedad, por otro lado, la propiedad raltive, es controloda mediante los valores de propiedad. Las démas propiedades dan diferentes efectos sobre los elementos, se pueden utilizar en función a las necesidades del sitio.

Propiedad Display

Cada elemento tiene un valor de visualización pre determinado según el tipo de elemento que sea.
Display: block;
Este siempre comienza en una nueva línea y ocupa todo el ancho disponible; elementos div, header, nav, section, footer, etc, son ejemplos de elementos con display: block;
Display: inline;
Este no comienza en una nueva línea y ocupa el tamaño necesario. Los elementos con display inline como los img, solo ocupan el tamaño de la imagen, mientras que figure al tener la propiedad block ocupan un espacio extra.
Display: inline-block;
No agrega un salto de línea, por lo que puede alinearse con otros elementos. Permite establecer el ancho y alto en la figura. Además se respetan margins, paddings.
Display: none;
Se usa comúnmente con JavaScript para ocultar elementos cuando el usuario realiza alguna opción como pulsar un botón. El resultado es que el elemento desaparece por completo de la página y no ocupa sitio.
Display: table;
Tiene una serie de displays diferentes para cada elemento de la tabla.

Notas

La propiedad display varia el valor de visualización de los elementos con dicha propiedad. Cada elemento en html cuenta con una propiedad de visualización predeterminada.

Resumen

Los elementos en HTML tienen un valor de visualización predeterminado segun el tipo de elemento que sea. La propiedad display, nos permite modificar como se relacionan los elementos en funcion de la propiedad. Los elementos div, header, nav, section, siempre comienzan en una nueva linea y ocupa todo el ancho disponible, la propiedad es display: block; La propiedad display: inline; ocupa solo el tamaño necesario. Las distintas propiedades tiene un efecto diferente en su valor de visualización.

Animación

Css3 permite animar la mayoría de los elementos. Para que una animación funcione se debe vincular a un elemento.
Div{
Animation-name: anim1;
}
@keyframes anim1
0%{width: 100px;
}
50% width: 500px;
100%{width: 500px;
}
Animation-duration: especifica cuanto tiempo deberá transcurrir entre animaciones. Si no se especifica, la animación quedara nula pues el tiempo predeterminado es de 0 segundos.
Animation-timing-function: especifica la curva de velocidad de la animación:
• Linear: especifica una animación con la misma velocidad-
• Ease: especifica una animación con inicio lento, luego rápido, y termina lento.
• Ease-in: inicia lento y termina rápido
• Ease-out: con un final lento

Aniation-direction: especifica si una animación debe reproducirse hacia adelante, hacia atrás o en ciclos alternos. No funciona si no se expresa animation-iteration: que especifica el número de veces que debe reproducirse la animación.
• Normal
• Reverse
• Alternate-reverse
Animation-delay: especifica el tiempo antes de ejecutar la animación. Toma de referencia el tiempo en el que cargo la página, si se especifica unidades negativas, estas se ejecutan antes de que la página cargue.
Animation-fill-mode: especifica un estilo para el elemento cuando esta no se esta reproduciendo (antes, después y ambas ocasiones)
• None: valor predeterminado. No aplicara ningún estilo.
• Forwards: retendrá los valores del estilo del ultimo fotograma clave (depende de animation-directio y iteration-)
• Backwards: obtendrá los valores de estilo establecidos en el primer fotograma, esto mientras la animación aún no se reproduce, una vez terminada la animación termina en el primer fotograma
• Both: no trabaja con ninguno de los valores ya sea iniciales o los finales.
Animation: propiedad rápida para configurar todas las propiedades de la animación: Nombre y duración son dos propiedades obligatorias

Animaciones con steps
La función steps() nos permite dividir una animación en segmentos en lugar de una transición a otra.

Notas

Si no se especifica el tiempo de la animación mediante animation-duration, la animación se anulará pues el valor de tiempo por default de las animaciones es de 0 segundos. Mediante keyframes se modifica el valor en porcentaje de la animación.

Resumen

CSS3 permite animar la mayoria de elementos utilizados. Para que las animaciones funciones deben estar vinculadas al elemento en cuestion. Por medio de los keyframes podemos darle distintos niveles de animacion, otorgandoles porcentaje en funcion al tiempo de la animación. Si no se especifica el tiempo de duración de la animación por medio de animation-duration la animación quedara nula, pues el valor de terminado es de 0 segundos.
En funcion de la propiedad de animacion este toma distintos tiempos, con animation-timing-function se especifica la curva de velocidad de la animación. Y así direction, reverse, delay, fill-mode, modifican cada uno en su función desde el tiempo de carga antes y despues de reproducir la animación, hasta el estilo inicial y final de esta.
La animación con steps permite dividir las animaciones en segmentos de fotogramas.

Referenciado

Con atributos id se utiliza el #, cuando es un elemento único, y el carácter, para elementos de tipo class.
Referenciado con pseudoclases
css3 incorpora nuevas pseudo clases que hacen la selección más específica, sin importar cuanto conocemos sobre sus atributos. La pseudo clase es usada agregando dos puntos luego de la referencia y antes de su nombre.
§ #referencia: ntch-child( )
Esta nos permite manipular un hijo específico. Los elementos referenciados por nth-child seleccionan todos los hijos relacionados, si no se especifica el elemento al momento de referenciarlo. Las palabras clave odd even. Nos permiten manipular y alternar todos los elementos referenciados por nth-child sin necesidad de crear nth-child para cada elemento.
§ #referencia: ntch-child(oven)
Existen otras pseudo clases relacionadas con esta última, la referencia first-child solo afecta al primer hijo, last-child afecta al último hijo, only-child solo afecta a un elemento cunado es hijo único. Otra pseudo clase importante es not ( ) los elementos referenciados con este hacen efecto sobre todos los elementos excepto con los elementos refrenciados entre los paréntesis
§ :not(.texto)
Selectores de tributo
Agregados recientemente estos nos permiten manipular el atributo dentro del elemento:
p[name^=”mi”]{… } donde los caracteres ^= , son las primeras dos letras del nombre del atributo referenciado.
La regla con el selector $=, afecta a los atributos cuyo nombre culmina con las letras referenciadas.
La regla con el selector *=, afecta a los atributos referenciados en el orden en que son escritos
Selectores de relación
Estos selectores utilizan los caracteres >, +, ¬ para especificar la relación entre los elementos. > Selecciona todos los elementos hijo de un elemento previamente referenciado div > p {… }.
El selector + selecciona los elementos que estén inmediatamente después de uno antes referenciado: nav + p {… }.
El selector con ¬ selecciona los elementos que son hermanos y que se encuentre después de este: p ¬ nav {… }.

Notas

Para referenciar con pseudoclases, es necesario referenciar al elemento que se quiera modificar, seguido de dos puntos y el atributo. Con el atributo ntch-child es posible modificar los hijos directos del elemento referenciado.

Resumen

CSS3 incorpora distintos selectores de atributo y pseudoclases, que permiten modificar especificamente el atributo del Elemento referenciado. Los selectores de relación utilizan caracteres especiales para establecer la relación que tiene el elment con otro.
